# Getting started

First install the package:
```bash
pip install django-gds-grabbage

# or

poetry add django-gds-grabbage
```

In your settings file, add the app to your `INSTALLED_APPS`:
```python
INSTALLED_APPS = [
    ...
    "django_gds_grabbage"
]
```
